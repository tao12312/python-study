# ---
파이썬 학습 보조 프로그램입니다.

#현재 구현된 기능
1.개념 모드
-개념이 정리된 글을 읽고 개념에 관한 문제를 풀 수 있습니다.
-개념 창에서 메뉴로 돌아갈수 있습니다.
2.학습 모드
-객관식과 주관식을 선택할 수 있습니다.
-객관식에선 정답시 초록빛, 틀릴시 빨간빛이 선지에 나옵니다.
-힌트를 볼 수 있습니다.
-주관식에서 코드 작성 후 실행시 파이썬이 실행 된 뒤 결과가 출력됩니다.
3.시험 모드
-한 창에 문제가 전부 나오는 시험 모드입니다.
-객관식 주관식이 전부 존재합니다.
-임시저장 기능이 존재합니다.
4.기록저징소
-틀린문제 맞는문제를 확인하고 다시 풀 수 있습니다.

#구현해야할 것.
1.llm 연결
-해당 문제는 고정된 문제와 답이라 ai의 다양한 문제가 출제 가능하게 답변 저장, json 형식에 맞춰 출력 받은 아웃 풋을 그대로 파싱해서 문제, 선지, 힌트 등에 입력을 하는 기능이 필요합니다.
2.json파일 연결 및 형식 구성
-json 파일을 받고 읽고 수정하는 기능이 필요합니다.
3.프롬프트 제작
-json형식을 강제하는 프롬(강제성 높아야함)과, 문제 출제 프롬프트(각 문제 형식에 따른), 피드백을 주는 프롬을 제작해야합니다.
4.UI개편
-단출한 UI를 좀더 세련되고 깔끔하게 만들어야합니다.
