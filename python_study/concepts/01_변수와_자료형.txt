[단원: 01. 변수와 자료형]

파이썬의 가장 기본적인 단위인 변수와 다양한 자료형에 대해 알아봅니다.

1. 변수 (Variables)
변수는 데이터를 저장하는 메모리 공간의 이름입니다. 파이썬에서는 변수 선언 시 타입을 명시하지 않고, 대입하는 값에 따라 타입이 자동으로 결정됩니다 (동적 타이핑).
- 예시: x = 10 (정수형 변수 선언)

2. 기본 자료형 (Data Types)
- 정수형 (int): 양의 정수, 음의 정수, 0을 나타냅니다.
- 실수형 (float): 소수점이 포함된 숫자를 나타냅니다.
- 문자열 (str): 따옴표로 감싸진 텍스트 데이터를 의미합니다.
- 불리언 (bool): 참(True)과 거짓(False)만을 가지는 논리 자료형입니다.

3. 복합 자료형 (Collections)
- 리스트 (List): 순서가 있고 수정 가능한 데이터 모음입니다. 대괄호 [ ]를 사용하여 선언합니다.
- 튜플 (Tuple): 순서가 있지만 수정 불가능한 데이터 모음입니다. 소괄호 ( )를 사용하여 선언합니다.
- 딕셔너리 (Dictionary): 키(Key)와 값(Value)의 쌍으로 이루어진 데이터 모음입니다. 중괄호 { }를 사용하여 선언합니다.
- 집합 (Set): 순서가 없고 중복을 허용하지 않는 모음입니다. 중괄호 { }를 사용하여 선언합니다.
