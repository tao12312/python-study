[단원: 02. 조건문과 반복문]

프로그램의 흐름을 제어하는 제어문에 대해 학습합니다.

1. 조건문 (if, elif, else)
조건에 따라 코드 블록의 실행 여부를 결정합니다. 파이썬에서는 들여쓰기(Indent)를 통해 블록을 구분하므로 들여쓰기 규칙이 매우 중요합니다.
- 예시:
  if score >= 90:
      print("A학점")
  elif score >= 80:
      print("B학점")
  else:
      print("C학점")

2. 반복문 (for, while)
- for문: 주로 범위(range)나 리스트 등의 시퀀스 데이터를 순회할 때 사용합니다.
  예시:
  for i in range(5):
      print(i) # 0부터 4까지 출력
- while문: 조건이 참(True)인 동안 계속해서 코드를 실행합니다. 조건이 거짓이 되도록 제어하지 않으면 무한 루프가 발생할 수 있습니다.
  예시:
  count = 0
  while count < 3:
      print(count)
      count += 1

3. break와 continue
- break: 반복문을 완전히 빠져나갑니다.
- continue: 현재 반복의 남은 코드를 건너뛰고 다음 반복으로 넘어갑니다.
